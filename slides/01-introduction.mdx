<ul>
  <Appear>

    </li>

      View usually draws something that <br /> users can see and interact with.
    </li>

# [ViewGroup](https://developer.android.com/reference/android/view/ViewGroup.html) 🤔

<ul>
  <Appear>
    <li>
      A ViewGroup is a special view that <br />can contain other views (called children.)
    </li>
    <li>
      The view group is the base class for <br /> layouts and views containers.
    </li>
    <li>
      LinearLayout, RelativeLayout, CoodinatorLayout, <br />ConstraintLayout, FrameLayout, etc.
    </li>
  </Appear>
</ul>

# [View](https://developer.android.com/reference/android/view/View.html) 💫

<ul>
  <Appear>
    <li>A View occupies a rectangular area on the screen <br /> and is responsible for drawing and event handling</li>
    <li>View is the base class for widgets, <br/>which are used to create interactive UI components</li>
    <li>Button, EditText, ImageView, TextView, etc.</li>
  </Appear>
</ul>
